# CLAUDE.md — Girl Code Couture

This repository holds marketing and campaign assets for **Girl Code Couture**.
Read this file before creating any content in this repo so everything stays on-brand.

## About the company

**Girl Code Couture** is a women's fashion and activewear brand founded by
Armina Swittenburg. It is more than apparel — it's a movement, built for the
woman who leads with confidence and lives with purpose. The brand celebrates
sisterhood: women hyping women, sharing their finds, and rising together.

- **Founder:** Armina Swittenburg (armina@swittenburg.com)
- **Master tagline:** *Discipline. Strength. Sisterhood.*
- **Campaign line:** *Coded to Elevate.*
- **Sign-off line:** *That's the code.*

### Brand pillars

1. **Built for Queens** — confidence is our standard.
2. **Performance Meets Style** — look good, do good.
3. **Community Over Competition** — we rise together.
4. **Exclusive. Empowering. You.** — that's the code.

## Brand colors

Primary palette (from the Girl Code Couture logo):

| Name | Hex | Usage |
|---|---|---|
| Couture Pink | `#EC008C` | Signature accent — logo script, CTAs, highlights. Never mute it. |
| Jet Black | `#0A0A0A` | Primary dark — dotted logotype on light, backgrounds on dark. |
| Pure White | `#FFFFFF` | Primary light — dotted logotype on dark, backgrounds on light. |

Secondary palette (from the "Coded to Elevate" activewear campaign):

| Name | Hex | Usage |
|---|---|---|
| Champagne Gold | `#C9A227` | Elevated/luxury accents — script headlines, icons, dividers. |
| Olive Legion | `#3F4224` | Grounding neutral — activewear tones, footer bands, backdrops. |

Rules: pink and gold are accents, never used together at equal weight in the
same lockup (pink = fashion/sisterhood energy, gold = luxury/discipline
energy). Body copy sits in black or white only.

## Logo

Two approved lockups (dotted rhinestone-style letterforms):

- **Light version:** black dotted "GIRL CODE" with pink dotted G + C, pink
  script "couture" — use on white/light backgrounds.
- **Dark version:** white dotted "GIRL CODE" with pink dotted G + C, pink
  script "couture" — use on black/dark backgrounds.

Never recolor the script "couture"; it is always Couture Pink.

## Voice and tone

- Confident, warm, girl's-girl energy — a best friend putting you on, never a
  corporation selling at you.
- First person, conversational, contractions, tasteful emoji in social copy.
- Empowering without clichés; specific over generic ("the slit hits right at
  the knee" beats "so flattering").
- Sisterhood framing: sharing, telling your girls, group chats, "girl code".

## Repo conventions

- Each marketing campaign lives in its own directory: `campaign/<campaign-name>/`.
- A campaign directory contains its own `README.md` brief, an `ads/` folder
  with one file per ad, exportable copy decks (CSV), and a `preview.html`
  mockup where useful.
- Use placeholders `[PRICE]` and `[SHOP LINK]` in ad copy until final values
  are supplied; keep them find-and-replace friendly.
- Facebook copy limits: hook within the first 125 characters of primary text,
  headlines ≤ 40 characters, descriptions ≤ 30 characters.
